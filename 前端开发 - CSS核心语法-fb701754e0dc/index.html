1. CSS 的介绍

  • 定义：全称Cascading Style Sheet，即层叠样式表，用于美化界面

2. CSS 的基本结构

  • css 由两个主要的部分构成

    • 选择器：用来选择标签
    • 样式规则：样式规则需要放到大括号里面，用来给选定标签添加样式

3. CSS 引入方式

1、行内式

  • 形式：直接在标签的 style 属性中添加 css 样式
  • 优点：方便、直观
  • 缺点：缺乏可重用性
  • 适用场景：几乎不用
